# DevToolbox (React Web Version)

现代化、全功能的开发工具箱。基于 React + TypeScript + Vite 构建。

## 功能特性
- 🚀 **极速体验**: 基于 Vite 构建，毫秒级热更新。
- 🎨 **高级美学**: 深色模式优先，玻璃拟态设计，流畅动画。
- 🛠️ **全方位工具**:
  - JSON 格式化/转义/类生成
  - 多进制转换 (Base16/32/58/62)
  - 核心加密 (AES, DES, RSA, 国密 SM2/3/4)
  - 文本处理 (对比, 正则, 转换)
  - 图片/二维码处理
  - 代码格式化 (Prettier 集成)

## 本地开发
1. 安装依赖:
   ```bash
   npm install
   ```
2. 启动开发服务器:
   ```bash
   npm run dev
   ```

## 部署到 Vercel
该项目已针对 Vercel 进行优化。
1. 将代码推送到 GitHub。
2. 在 Vercel 中导入项目。
3. 框架选择 **Vite**。
4. 构建命令 **npm run build**。
